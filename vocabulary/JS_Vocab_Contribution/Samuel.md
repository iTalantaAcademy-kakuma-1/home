`Javascript Terminologies`

1. The trim() 
   
   method removes whitespace from both sides of a string

2. The concat()
   
   Method can be used instead of the plus operator

3. Function
   
   A keyword that allows to define your own functions in Javascript

4. Loop
   
   Used to perform same action multiple times in a row

5. String slice()
   
   Extracts a part of a string and returns the extracted part in a new string.
